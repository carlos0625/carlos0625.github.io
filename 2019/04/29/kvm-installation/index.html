发表于 2019-04-29

前言

本文虚拟机安装Ubuntu 18.04 LST，KVM为Ubuntu 14.04

如何安装虚拟机，请自行百度或谷歌。要求虚拟机的配置为：

  • 4个VCPU

  • 2GB的内存

  • 50GB虚拟磁盘(重新编译内核需要70G以上)

  • 开启Virtualize Intel VT-x/EPT and AMD-V/RV 和 Virtual CPU performance counters
